# サーバとは？

「リクエスト（お願い）が来たら、何かを返すプログラム」

---

## 例

ブラウザが「このページちょうだい」

アプリが「このデータ保存して」

→ それに答える役

---

## ポイント

👉 パソコンの中で動いていてもサーバ  
👉 インターネット公開してなくてもサーバ

なので  
**あなたが使っている Express も Vite も、どちらもサーバです。**

---

## 開発時の通信イメージ

```text
ブラウザ
  │
  │  http://localhost:5173
  ▼
Vite サーバ（画面を返す・開発用）
  │
  │  /api の通信だけ
  ▼
Express サーバ（処理・DB）

```
## ① Express サーバ（本体・エンジン役）
何をしている？

-データを保存・取得する
-ログイン状態を管理する
-SQL Server と通信する

「結果」を返す

例（人に説明するなら）

Expressは  
「仕事をする係」  
データを考えたり、計算したり、保存したりします。

特徴

速さより 正確さ・安定性 が重要

画面の見た目は扱わない

24時間止まらない前提
